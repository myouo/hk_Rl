# PRD

The product requirements document lives at the repository root:
[`../hollow_knight_rl_prd.md`](../hollow_knight_rl_prd.md).

It is the authoritative product vision. The documents in this `docs/` folder
are the *engineering specifications* that implement it — each cross-references
the relevant PRD section. When PRD and a spec disagree, treat it as a bug:
reconcile and record the decision in an [ADR](./adr/).
